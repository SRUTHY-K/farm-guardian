# Crop Disease, Pest, and Weed Diagnosis Skill

Use this skill when analyzing a crop image or text symptoms:
1. Identify the crop type (e.g., tomato, rice/paddy, wheat, maize).
2. Examine visual abnormalities:
   - Yellow/brown spots, powdery patches, or rust.
   - Wilting, leaf curling, or stem rot.
   - Pest signs (aphids, caterpillars, stem borers).
   - Weed proliferation (e.g., broadleaf, grass weeds).
3. Diagnose the issue:
   - Identify the likely pathogen (fungal, bacterial, viral) or pest/weed type.
   - Provide confidence score (e.g., "High (90%)", "Moderate").
4. Recommending treatment:
   - Prioritize organic/biological treatments first (e.g., neem oil, copper hydroxide, pruning).
   - Suggest chemical fungicides/pesticides only if severity is high.
   - **Important**: Must include safety instructions (protective gear, proper dilution) and request human confirmation for chemical treatments.
