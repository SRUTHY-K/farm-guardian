# Fertilizer Schedule & Soil Advisor Skill

Use this skill when suggesting nutrient inputs:
1. Recommends nitrogen (N), phosphorus (P), and potassium (K) ratios based on:
   - Crop type (e.g., tomatoes need high potash during fruiting, paddy needs nitrogen for vegetative stage).
   - Soil type (clayey, sandy, loamy).
   - Crop growth stage (seedling, vegetative, flowering, fruiting).
2. Calculate total fertilizer needed for the farmer's acreage.
3. Suggest organic alternatives (manure, compost, bio-fertilizers) to improve soil health.
4. Warn about excess nitrogen (which makes plants prone to pests) and chemical runoff.
